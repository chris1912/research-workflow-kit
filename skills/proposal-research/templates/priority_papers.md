# 优先全文文献 / Priority Papers

Grok annotation: Template created by Grok on 2026-07-20 for OA-first priority full-text handling.

## Selection rule / 选择规则

Select **3–8** decision-critical papers from `MERGED_CORE_PAPERS.md` (or from `core_papers.md` only before the authority transition). Prefer papers that:

- anchor the main scientific claim or gap;
- define methods the proposal will adapt or improve;
- supply counterevidence or risk that must not be ignored;
- are more likely to have lawful open-access full text.

Do **not** attempt paywall bypass. Lawful routes only: publisher OA, repository OA, author-accepted manuscript when permitted, institutional access already available to the user, or manual library request.
