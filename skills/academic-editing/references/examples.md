# Examples

## Academic Polishing

User:

```text
我是做医学影像分割的，请帮我把下面这段话润色成 SCI 论文风格。
```

Expected output:

- polished English or Chinese text, depending on request;
- concise revision notes;
- terminology suggestions;
- uncertainty notes for missing validation details.

## CN to EN Translation

User:

```text
请将以下中文段落翻译成适合 SCI 论文的英文表达，并给出术语表。
```

Expected output:

- natural academic English;
- terminology table;
- translation notes;
- no invented data or references.

## Abstract with Missing Results

User:

```text
请根据方法描述写摘要，但我还没有实验结果。
```

Expected output:

- abstract draft with placeholders for missing evidence;
- no fabricated numerical results;
- missing information section.

## Reviewer Response

User:

```text
Reviewer: The novelty of the proposed method is unclear.
```

Expected output:

- polite acknowledgement;
- response draft;
- proposed manuscript revision statement;
- no claim that actual revisions were made unless the user says so.
