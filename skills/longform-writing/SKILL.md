# Long-form Writing

Codex annotation: Public alias prepared by Codex on 2026-07-15.

Read `docs/START_HERE.html` and announce the selected route. Use the reusable modules under `modules/` for outline design, section drafting, evidence placement, and quality review.

## Method

1. Define the audience, claim scope, genre, required sections, and evidence standard.
2. Build a section map before drafting prose.
3. Draft one section at a time, attaching sources and uncertainty to each substantive claim.
4. Check transitions, terminology, citation integrity, and compliance with the target venue.
5. Hand the result to `academic-editing` and `prose-lint`, then request human review.

Never fabricate evidence, citations, quotations, experiments, or reviewer feedback. Keep generated drafts and exports outside tracked source folders. Include `docs/START_HERE.html` in phase summaries.
